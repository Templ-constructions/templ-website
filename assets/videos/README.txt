VIDEO FILES GO HERE
===================

Compress these three source videos in HandBrake (handbrake.fr) before
dropping them in this folder. Settings:

  Format:      MP4
  Video Codec: H.264
  Quality:     RF 23 (constant quality)
  Framerate:   30 fps
  Resolution:  1920x1080 maximum
  Audio:       None (remove the audio track — all three are muted)
  Target:      under ~15 MB each

Source file                    ->  Save into this folder as
-------------------------------    -------------------------------
TC.March.OxfordSt.PH.mp4       ->  hero-homepage.mp4
TC.Rhyll-HLR-2.mp4             ->  hero-about.mp4
TC.Feb.BRA.2.mp4              ->  gallery-mount-eliza.mp4

Do NOT use TC.Feb.BRA.4.mp4 — excluded.

The site references these exact filenames, so the names must match.
Until the files are here, the video areas will simply show black —
nothing breaks.
